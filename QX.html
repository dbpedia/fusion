Description

This is a prototype created for the following grant proposal: CrossWikifact . We used DBpedia to compare the values of the different Infoboxes of 128 Wikipedia versions to Wikidata in order to help editors make Wikipedia/Wikidata better. The proposal is about elaborating this website in the following way:

  • add references which are cited in Wikipedia
  • add links to the Wikipedias/Wikidata, so users can easily edit
  • at the moment data a lot of available data from DBpedia and Wikidata is still missing on this website (e.g. geocoordinates)
  • if possible, simplify the import into wikidata, e.g. by pre-filled forms
  • extensive details are given in the proposal
We (DBpedia) plan to deliver the following things as in-kind contribution (i.e. no covered by the requested amount):
  • provide weekly updates for all infoboxes and updates for wikidata in realtime
  • we would like to better check correctness of the values in the table
  • add more potential values from more official datasources (governments, libraries) to be able to compare more values
  • improve the alignment between the infoboxes and wikidata
